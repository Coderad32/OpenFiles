## Connect | Command | system menu
Connect/ the command/ system/ menu/

```perl
import system command
use system command
use english
BEGIN: <Application> DATA | META | Command
  # This is the application DATA for the META Command Interface
  # 0 exit or reset
  # 1 command tools
  # 2 update system
  # 3 upgrade system command menu
  #   ###
return 1
exit 0
END
```
## Output

```perl

<META::RESULT>
  selected: 1
  meaning: command tools
  status: EXIT_SUCCESS
</META::RESULT>

```
